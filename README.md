Notes Saver Application (Python + MySQL)
 
This is a console-based Contact Saver Application developed using Python and MySQL.
 
## Features
- Add new Notes
- View all 
- Search 
- Update 
- Delete 
 
## Technologies Used
- Python
- MySQL
- mysql-connector-python
 
## Files in this Project
- Notes_saver.py → Python application
- Notes_saver.sql → MySQL queries to create database and table
